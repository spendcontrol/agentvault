Features

Built for trust, not promises

Multi-Token Treasury

USDC, WETH, DAI, stETH -- all in one vault. Your agent has a unified treasury.

Per-Token Limits

Set daily and per-tx limits for each token separately. Granular, on-chain controls.

Expense Reports

Every spend recorded on-chain with a reason. Full audit trail, immutable and transparent.

For Developers

Integrate in minutes

Connect your agent to its vault using MCP, Python SDK, a skill file, or direct CLI calls.

// claude_desktop_config.json
{
  "mcpServers": {
    "spendcontrol": {
      "command": "npx",
      "args": ["spendcontrol-mcp"],
      "env": {
        "VAULT_ADDRESS": "0xYourVault...",
        "AGENT_PRIVATE_KEY": "0x...",
        "RPC_URL": "https://sepolia.base.org"
      }
    }
  }
}
from spendcontrol import SpendControlClient

client = SpendControlClient(
    vault_address="0xYourVault...",
    private_key="0x...",
    rpc_url="https://sepolia.base.org"
)

# Check available budget
budget = client.remaining_daily_budget(token="USDC")
print(f"Daily budget remaining: {budget} USDC")

# Spend with a reason
tx = client.spend(
    token="USDC",
    to="0xVendor...",
    amount="10.00",
    reason="API subscription - March 2026"
)
// .well-known/ai-plugin.json
{
  "schema_version": "v1",
  "name": "SpendControl",
  "description": "Treasury for AI agent spending",
  "api": {
    "type": "openapi",
    "url": "https://spendcontrol.xyz/openapi.yaml"
  },
  "auth": {
    "type": "none"
  }
}
# Install the CLI
$ npm install -g spendcontrol-cli

# Check vault balance
$ spendcontrol balance --vault 0xYourVault --token USDC
Balance: 5,000.00 USDC

# Spend from the vault
$ spendcontrol spend \
    --vault 0xYourVault \
    --token USDC \
    --to 0xVendor \
    --amount 25.00 \
    --reason "Cloud compute - batch job #4291"

# View recent expenses
$ spendcontrol expenses --vault 0xYourVault --last 10
