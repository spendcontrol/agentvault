How It Works

Four steps to autonomous agent finance

Go from zero to a fully funded AI agent treasury in under a minute.

1

Create a Vault

Deploy your agent's personal treasury in one click. Each vault is an independent smart contract.

2

Fund It

Deposit USDC, WETH, or any ERC20 token. Optionally stake ETH via Lido for passive yield.

3

Agent Operates

Your agent spends within daily and per-transaction limits. Every spend has a recorded reason.

4

Full Transparency

Expense reports on-chain. Pause anytime. Withdraw anytime. You stay in control.

Why AgentVault

Built for the agentic economy

Multi-Token Treasury

Deposit any ERC20 token: USDC, WETH, DAI, and more. Your agent has a single vault for all its assets.

On-Chain Spending Rules

Daily limits, per-transaction caps, recipient whitelists. All enforced by the smart contract, not a backend.

Expense Reports

Every spend is recorded with a reason, timestamp, and recipient. Full auditability, immutable and on-chain.

For Developers

Integrate in minutes

Connect your agent to its vault using MCP, Python SDK, a skill file, or direct CLI calls.

// claude_desktop_config.json
{
  "mcpServers": {
    "agentvault": {
      "command": "npx",
      "args": ["agentvault-mcp"],
      "env": {
        "VAULT_ADDRESS": "0xYourVault...",
        "AGENT_PRIVATE_KEY": "0x...",
        "RPC_URL": "https://sepolia.base.org"
      }
    }
  }
}
from agentvault import AgentVaultClient

client = AgentVaultClient(
    vault_address="0xYourVault...",
    private_key="0x...",
    rpc_url="https://sepolia.base.org"
)

# Check available budget
budget = client.remaining_daily_budget(token="USDC")
print(f"Daily budget remaining: {budget} USDC")

# Spend with a reason
tx = client.spend(
    token="USDC",
    to="0xVendor...",
    amount="10.00",
    reason="API subscription - March 2026"
)
// .well-known/ai-plugin.json
{
  "schema_version": "v1",
  "name": "AgentVault",
  "description": "Treasury for AI agent spending",
  "api": {
    "type": "openapi",
    "url": "https://agentvault.xyz/openapi.yaml"
  },
  "auth": {
    "type": "none"
  }
}
# Install the CLI
$ npm install -g agentvault-cli

# Check vault balance
$ agentvault balance --vault 0xYourVault --token USDC
Balance: 5,000.00 USDC

# Spend from the vault
$ agentvault spend \
    --vault 0xYourVault \
    --token USDC \
    --to 0xVendor \
    --amount 25.00 \
    --reason "Cloud compute - batch job #4291"

# View recent expenses
$ agentvault expenses --vault 0xYourVault --last 10
